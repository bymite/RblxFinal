# RobloxProxy Tweak

Forces Roblox to route all traffic through `mainline.proxy.rlwy.net:55396`.

## Setup

1. Create a new **private** GitHub repo
2. Upload all these files to it
3. Create an `ipa/` folder in the repo and add your `Roblox.ipa` file renamed to `Roblox.ipa`
4. Push to `main` — GitHub Actions will build automatically
5. Go to **Actions** tab → click the latest run → download `Roblox_patched.ipa` from Artifacts
6. Sideload with **Sideloadly** or **AltStore**

## Note
Make the repo **private** since it contains the IPA.
